O que é a internet?

A internet é uma rede de computadores que permite a comunicação entre dispositivos conectados a ela. Ela é composta por uma rede de servidores e computadores que compartilham informações e recursos entre si. A internet é usada para acessar informações, enviar e receber mensagens, fazer compras online, entre outras coisas.

O que é HTML?

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar páginas da web. Ele é usado para estruturar e organizar o conteúdo de uma página da web, incluindo texto, imagens, links e outros elementos. O HTML é usado para criar a estrutura básica de uma página da web, e é interpretado pelos navegadores para exibir o conteúdo da página.

Tags de Texto

As tags HTML são elementos que são usados para identificar e estruturar o conteúdo de uma página da web. Elas são representadas por palavras entre angle brackets (< e >). Por exemplo, a tag <p> é usada para criar um parágrafo de texto.

Tags de Lista

Tags de Lista são aquelas destinadas a organizar um texto em forma de listagem. Os itens dessa lista podem ser ordenados e não ordenados.

Listas não ordenadas são compostas pela tag <ul> </ul>", com os itens aninhados nas tag <li> </li>, conforme o exemplo:

Estrutura:

<ul>

<li> Item 1 </li>

<li> Item 2 </li>

<li> Item 3 </li>

</ul>

Saída:

  • Item 1
  • Item 2
  • Item 3

O mesmo é válido para listas ordenadas, porém, nesses casos a lista é representada pela tag <ol>, conforme o exemplo abaixo:

Estrutura:

<ol>

<li> Item 1 </li>

<li> Item 2 </li>

<li> Item 3 </li>

</ol>

Saída:

  1. Item 1
  2. Item 2
  3. Item 3

Tags de Input

Tags de Input são aquelas destinadas a capturar informações do usuário. Elas podem ser de vários tipos, como:

  • Input de texto
  • Input de número
  • Input de data
  • Input de e-mail

As tags de input são usadas para criar campos de entrada de dados no formulário. Elas são representadas pela tag <input> e podem ter vários atributos, como:

  • type: define o tipo de input
  • name: define o nome do campo
  • placeholder: define o texto de ajuda
  • required: define se o campo é obrigatório

Exemplo de uso:

<input type="text" name="nome" placeholder="Digite seu nome " required>

Tags de Link

Tags de link são aquelas destinadas a criar links para outras páginas ou recursos. Elas são representadas pela tag <a> e podem ter vários atributos, tais como:

  • href: define a URL do link
  • title: define o título do link
  • target: define a janela de exibição do link

Exemplo de uso:

<a href="https://www.google.com" title="Ir para o Google" target="_blank">Ir para o Google</a>

Essa tag de link é para a URL "https://www.google.com", com o título "Ir para o Google" e a janela de exibição aberta em uma nova janela, conforme o exemplo abaixo:
